# .emacs.d

I'm using `Centaur Emacs` for a few years, is time to building my own configured for emacs.

## Features

* Using lsp-bridge as the only lsp client
* Only support Go/Rust/Python program languages
* Only support linux
* ...

## Install

```sh
mv ~/.emacs.d ~/.emacs.d.bak
git clone https://github.com/mowind/.emacs.d ~/.emacs.d
```

## Update

`git pull`
